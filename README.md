# 🖐️ Hand Volume Control

A computer vision project that enables touch-free system volume control using real-time hand gesture recognition. The application tracks hand landmarks through a webcam and adjusts the system volume based on the distance between the thumb and index finger.

---

## 📌 Project Overview

Hand Volume Control is a real-time gesture recognition application built with Python, OpenCV, and MediaPipe. It detects hand movements using a webcam and maps the distance between the thumb and index finger to the system's volume level, providing an intuitive, contactless user experience.

---

## 🚀 Features

- 🎥 Real-time hand tracking using a webcam
- ✋ Detects hand landmarks with MediaPipe
- 🔊 Controls system volume using finger distance
- 📊 Displays volume level on screen
- ⚡ Fast and responsive gesture recognition
- 💻 Simple and user-friendly interface

---

## 🛠️ Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy
- Pycaw (Windows Audio Control)
- ctypes

---

## 📂 Project Workflow

1. Capture live video from the webcam.
2. Detect hand landmarks using MediaPipe.
3. Measure the distance between the thumb and index finger.
4. Convert the distance into a volume level.
5. Update the system volume in real time.
6. Display the current volume percentage on the screen.
